**Document ID:** HB-CHAT-001
**Status:** Final

---

# Purpose

HAT-BAZAR Chat System এমনভাবে তৈরি করা হয়েছে যাতে Customer এবং Seller সহজে, নিরাপদে এবং দ্রুত যোগাযোগ করতে পারে।

---

# Customer & Seller Chat

Customer যেকোনো Shop-এর Seller-এর সাথে Chat করতে পারবে।

Product কেনার আগে এবং পরে Chat করা যাবে।

---

# First Message Rule

- Customer প্রথম Message পাঠাবে।
- Seller আগে থেকে কোনো Customer-কে Message পাঠাতে পারবে না।
- Customer Chat শুরু করলে Seller Reply দিতে পারবে।

---

# Chat Features

Chat-এ থাকবে:

- Text Message
- Product Link
- Image
- Emoji

থাকবে না:

- Voice Call
- Video Call
- SMS
- Email
- Location Share
- PDF বা অন্যান্য File Upload

---

# Offline Chat

- Customer Offline থাকলেও Seller Reply দিতে পারবে।
- Seller Offline থাকলেও Customer Message পাঠাতে পারবে।
- Online হলে সব Message দেখা যাবে।

---

# Message Status

প্রতিটি Message-এর Status থাকবে:

- Sent
- Delivered
- Seen

---

# Product Sharing

Seller নিজের Shop-এর Product Chat-এ Share করতে পারবে।

Product Card-এ থাকবে:

- Product Image
- Product Name
- Price
- Rating
- View Product Button

Customer View Product চাপলে সরাসরি Product Page খুলবে।

---

# Order Sharing

Seller Customer-এর Order Share করতে পারবে।

Order Card-এ থাকবে:

- Order ID
- Shop Name
- Order Status
- Order Date
- View Order Button

Customer View Order চাপলে Order Details খুলবে।

---

# AI Smart Reply

AI Seller-কে উত্তর সাজেস্ট করবে।

Seller চাইলে:

- Send
- Edit
- Ignore

করতে পারবে।

AI কখনো নিজে থেকে Message পাঠাবে না।

---

# AI Translation (Future)

Version 2.0-এ

AI বিভিন্ন ভাষার Message অনুবাদ করবে।

Original Message সবসময় সংরক্ষিত থাকবে।

---

# Chat Notification

নতুন Message এলে:

- Push Notification যাবে।
- Sender Name দেখা যাবে।
- Message Preview দেখা যাবে।
- Notification চাপলে Chat খুলবে।

---

# Chat Archive

Customer এবং Seller Chat Archive করতে পারবে।

Archive করলে:

- Chat Delete হবে না।
- Inbox থেকে সরে যাবে।
- নতুন Message এলে আবার Inbox-এ ফিরে আসবে।

---

# Chat History

- সব Chat History স্থায়ীভাবে সংরক্ষিত হবে।
- Server থেকে Chat Delete করা যাবে না।

---

# Business Rules

- Customer আগে Chat শুরু করবে।
- Seller নিজে থেকে Promotional Message পাঠাতে পারবে না।
- Product এবং Order সরাসরি Chat-এ Share করা যাবে।
- সব Chat নিরাপদভাবে সংরক্ষিত হবে।
- সব Chat Audit Log-এ সংরক্ষণ করা হবে।
