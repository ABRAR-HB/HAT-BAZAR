# Purpose

HAT-BAZAR Search System এমনভাবে তৈরি করা হয়েছে যাতে User দ্রুত, নির্ভুল এবং সহজে Product, Shop এবং Seller খুঁজে পায়।

---

# Smart Search

User Search করতে পারবে:

- Product Name
- Shop Name
- Seller ID
- Category
- Brand
- Location (Future)

---

# Search Results

## Product Search

Product Name অনুযায়ী মিল থাকা Product দেখানো হবে।

সবচেয়ে প্রাসঙ্গিক Product আগে দেখানো হবে।

---

## Shop Search

Shop Name লিখলে সেই Shop প্রথমে দেখানো হবে।

তারপর সেই Shop-এর Product দেখানো যাবে।

---

## Seller ID Search

Seller ID লিখলে সরাসরি সেই Seller-এর Shop দেখানো হবে।

---

# Unique Rules

- প্রতিটি Shop Name Unique হবে।
- একই Shop Name দিয়ে দ্বিতীয় Shop খোলা যাবে না।
- প্রতিটি Seller ID Unique হবে।

---

# AI Logo Verification

নতুন Shop তৈরি করার সময়:

- AI Shop Logo পরীক্ষা করবে।
- Logo আগের কোনো Shop-এর সাথে অত্যন্ত মিল থাকলে Shop Auto Approve হবে না।
- Admin Review করবে।

---

Available Filters:

# AI Smart Search

AI:

- বানান ভুল বুঝবে।
- বাংলা ও ইংরেজি Search বুঝবে।
- সমার্থক শব্দ বুঝবে।
- সবচেয়ে প্রাসঙ্গিক Result দেখাবে।

---

# Search Ranking

Ranking নির্ধারণ হবে:

- Search Relevance
- Rating
- Sales
- Reviews
- Popularity
- User Filter

Sponsored Product থাকলে:

Sponsored Tag দেখানো হবে।

---

# Search History

User দেখতে পারবে:

- Recent Searches

User পারবে:

- একটি Search Delete করতে।
- সব History Delete করতে।
- Search History Disable করতে।

---

# Business Rules

- Search Result দ্রুত দেখাতে Indexing ব্যবহার করা হবে।
- AI Search Ranking উন্নত করবে।
- Search History শুধুমাত্র User দেখতে পারবে।
- Sponsored Product অবশ্যই Sponsored হিসেবে দেখাতে হবে।
- Search Filter একসাথে একাধিক ব্যবহার করা যাবে।
